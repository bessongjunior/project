# Web Module (SmartPick Simulation)
A full-stack dashboard for visualizing and simulating real-time delivery route optimization.

## Backend
- **FastAPI**: Serving model inference.
- **Services**: Inference engines for the ST-GCN/T-GCN models.

## Frontend
- **React (TypeScript)**: Interactive map and task dashboard.
- **Leaflet/OSM**: Road-snapped visualization using OSRM.
